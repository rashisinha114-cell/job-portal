# Job Portal

A web-based job portal that allows users to search and apply for jobs and enables recruiters to post job openings.

## Features
- User registration and login
- Search for jobs
- Apply for jobs
- Recruiters can post job listings
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Flask
- MYSQL


## Future Improvements
- Add job recommendation system
- Add email notifications
- Improve UI design
